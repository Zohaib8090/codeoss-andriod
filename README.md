# CodeOSS Android

A powerful, native Android IDE built for developers who need VS Code-like power on the go. CodeOSS Android integrates a real-time Open VSX Extension Marketplace, a fully functional Terminal with Node.js support, and a premium, responsive UI.

## 🚀 Key Features

- **Open VSX Marketplace**: Browse and install thousands of VS Code extensions directly within the app.
- **Deep Extension Integration**: Full extension detail pages with icons, descriptions, download counts, and real-time search.
- **Native Node.js Runtime**: Run `npm`, `node`, and other tools natively on Android.
- **Premium UI/UX**: Modern glassmorphism design, smooth micro-animations, and a highly responsive sidebar.
- **Project Management**: Isolated project environments with a dedicated file explorer.
- **Real Terminal**: Full PTY support with gesture-based interactions.

## 🛠️ Recent Updates

- **Real-time Marketplace**: Switched from placeholders to a live integration with [Open VSX](https://open-vsx.org/).
- **Official Extension Priority**: Enhanced search algorithm to prioritize official publishers (like Google's Gemini Code Assist).
- **UI Overhaul**: Redesigned the activity bar and sidebars with a premium, IDE-grade aesthetic.
- **Notification Permissions**: Automatic handling of Android 13+ notification permissions on startup.

## 📱 Getting Started

1. Clone the repository.
2. Open in Android Studio.
3. Build and deploy to an Android 16+ device (compatible with Android 11+).
